UTM Codes Or Parameters

What is UTM Code or Parameter?

UTM is an standing for Urchin Tracking Module and it's the format which Google uses to track URL's.

UTM Parameters are simple small pieces of information that can be added to the end of any URL that gives Google information about that link.

Example - https://www.mywebsite.com/?utm_source=emailcampaign

The part after the question mark is a UTM Parameter. When that link is clicked, Google Analytics for that particular website will read & record that parameter information.

UTM Parameters

5 UTM parameters that can be used for tracking diffrent pieces of information in Google Analytics

1. (utm_source) Campaign Source 2. (utm_medium) Campaign Medium 3. (utm_campaign) Campaign Name 4. (utm_term) Campaign Term 5. (utm_content) Campaign Content
